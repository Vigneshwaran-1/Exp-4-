# Exp - 4 HOSPITAL MANAGEMENT SYSTEM

# AIM:
To study the problem statement, SRS document and draw all the UML diagrams of a Hospital Management System.

# SRS (Procedure):
~~~
Problem: Hospitals need a system to manage patients, doctors, appointments, treatments, and billing.
Actors:
Admin, Doctor, Receptionist, Patient.

Modules:
Patient → Register, update, search
Doctor → Add, assign, schedule
Appointment → Book, cancel
Treatment → Diagnosis, prescription
Billing → Generate invoice
Non‑Functional: Easy to use, secure, reliable, fast.
~~~
